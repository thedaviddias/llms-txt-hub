# atari-style

Terminal-based interactive games, creative tools, and visual demos inspired by classic Atari aesthetics.

## Key Features

- Classic arcade games (Pac-Man, Galaga, Breakout, Grand Prix)
- ASCII Painter creative tool
- Visual demos (starfield, screen saver, platonic solids)
- Full joystick and keyboard support
- GPU-accelerated shader pipeline with video export

## About llms.txt Implementation

The llms.txt provides comprehensive documentation of all games, tools, and demos with quick-start commands, CLI references, and cross-project integration points within the jcaldwell-labs terminal ecosystem.
