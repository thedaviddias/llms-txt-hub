# DondeGo

DondeGo is a public local event discovery API for Barcelona and Madrid. The site publishes a machine-readable discovery package: `llms.txt`, OpenAPI 3.1 (`/openapi.json`), an API Catalog linkset (`/.well-known/api-catalog`), a rich discovery JSON (`/api-catalog.json`), public Postman documentation, and a documentation-only Agent Skills index (`/.well-known/agent-skills/`).

The Agent Skill is documentation-only: no scripts, no executable code, no MCP server, no OAuth issuer, and no WebMCP tools.

Attribution: any public website, app, newsletter, bot, AI assistant, or partner integration that displays DondeGo event descriptions or substantial DondeGo event data must include a visible direct link to the canonical DondeGo.es event page as the source.
