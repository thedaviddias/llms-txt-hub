# VibeRaven

VibeRaven is a production-readiness gate for AI-built applications. It helps
coding agents check launch readiness before deployment, real users, auth,
billing, database changes, Supabase RLS, environment variables, webhooks,
monitoring, rate limits, or tests.

## Key Focus Areas

- AI-built app production readiness
- Agent tasklists and launch gates
- Supabase RLS, Vercel deploy, auth, billing, and webhook checks
- `npx -y viberaven --agent-mode`

## About llms.txt Implementation

VibeRaven publishes `llms.txt` and `llms-full.txt` files for coding agents that
need production-readiness context, commands, and launch-gate rules.
