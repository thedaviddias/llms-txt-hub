# Crypto APIs REST

Machine-readable entry point served directly from the Crypto APIs production REST host, https://rest.cryptoapis.io — the base URL that applications and AI agents call. One integration covers 20+ blockchains across UTXO, EVM, Solana, XRP, Kaspa, Tezos, and more.

Covers Address Latest, Address History, Block Data, Transactions Data, HD Wallets, Blockchain Fees, Prepare Transactions, Broadcast Transactions, Simulate Transactions, Contracts, Blockchain Events, Market Data, Utils, and AML address screening. Includes authentication and pagination conventions, representative endpoint paths, a pointer to the official Crypto APIs MCP server at ai.cryptoapis.io/mcp, and explicit guardrails directing agents to the current API surface while marking legacy v1 and retired SDK docs as deprecated.
