# claude-scaffold

npx CLI for deploying a complete Claude Code workflow to any repository. Deploys CLAUDE.md, hooks, skills, and agents in one command. 18 domain skills auto-activate via UserPromptSubmit hooks based on project context (FastAPI, LangGraph, RAG, MLflow, etc.). Cross-repo sync via `npx claude-scaffold update --all`.
