# Kernel-Error

Kernel-Error is a German technical blog by IT security expert Sebastian van de Meer, running since 2003 with over 600 articles.

## Key Topics

- IT Security: Penetration testing, system hardening, security research
- Mail Infrastructure: Postfix, Dovecot, SPF, DKIM, DMARC, DANE
- DNS & DNSSEC: BIND configuration, DoT/DoH
- Unix/BSD/Linux: FreeBSD administration (main focus)
- Self-Hosting: ZFS, Jails, high availability
- IoT & Embedded Systems: Smart device security, microcontrollers
