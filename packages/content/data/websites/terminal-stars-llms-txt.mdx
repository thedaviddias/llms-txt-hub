# terminal-stars

Educational 3D starfield visualizer demonstrating terminal graphics programming concepts in C.

## Key Features

- 3D perspective projection with camera transforms
- Six visual effects (linear, spiral, warp, tunnel, explode, wave)
- Double-buffered flicker-free rendering
- Delta-time frame-rate independent animation
- CLI options for stars, speed, effect, and zoom

## About llms.txt Implementation

The llms.txt provides learning path through key files, explains core concepts (projection math, camera transforms), and links to related jcaldwell-labs terminal graphics projects.
