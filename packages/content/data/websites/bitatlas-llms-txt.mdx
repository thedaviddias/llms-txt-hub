# BitAtlas

Zero Knowledge Cloud Drive for Humans and Agents. BitAtlas is agent-native encrypted cloud storage — all files are encrypted client-side with AES-256-GCM before upload. The server never sees plaintext data or encryption keys. Includes an MCP server for AI agent integration.
