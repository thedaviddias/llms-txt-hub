# Ariadne Loop

Ariadne Loop helps Codex, Claude Code, OpenClaw, and other coding agents work from a bounded loop contract instead of a loose prompt.

## Key Focus Areas

- Loop Engineering specs for coding agents
- Agent packets with verifier gates, stop rules, rollback rules, and budgets
- JSON report contracts for inspect-act-verify-decide cycles
- Local-first browser builder and command-line workflow

## About llms.txt Implementation

Ariadne Loop publishes an `llms.txt` file that gives AI agents a compact index of the project, browser builder, templates, OpenClaw guide, verifier recipes, and coding-agent workflow.
