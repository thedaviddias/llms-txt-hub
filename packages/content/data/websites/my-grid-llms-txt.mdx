# my-grid

Spatial ASCII canvas editor transforming terminals into infinite 2D workspaces with vim-style modal editing.

## Key Features

- Infinite sparse canvas for ASCII art and diagrams
- Dynamic zones (PTY terminals, watch commands, sockets)
- Vim-style modal editing (NAV, EDIT, PAN, VISUAL, DRAW)
- Workspace layouts saved as YAML templates
- API server for external process control

## About llms.txt Implementation

The llms.txt documents zone types, command reference, AI integration via socket zones, and use cases for DevOps dashboards and ASCII diagramming.
