# Crypto APIs

Crypto APIs Inc. provides a blockchain infrastructure suite offering unified APIs for blockchain data, transaction processing, wallet management, blockchain events, and crypto market data. Newer additions include AML address screening (Verify Address) and Model Context Protocol servers plus an n8n MCP client for AI-agent integrations. Coverage spans 20+ blockchains.
