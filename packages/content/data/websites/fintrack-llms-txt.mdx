# fintrack

Terminal-based personal finance tracking following Unix philosophy principles with local PostgreSQL storage.

## Key Features

- Account management (checking, savings, credit, cash, investment, loan)
- Transaction tracking with categories, payees, tags, and notes
- Hierarchical category system
- JSON output for scripting and automation
- Cross-platform Go binary

## About llms.txt Implementation

The llms.txt provides complete command reference, architecture overview, and integration patterns for developers building finance-related terminal workflows.
