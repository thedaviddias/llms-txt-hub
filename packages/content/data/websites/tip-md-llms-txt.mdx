# tip.md

> Tip.md enables developers to receive cryptocurrency tips (Ethereum, Base, Solana, Bitcoin Lightning) via a simple button embeddable in GitHub READMEs, websites, or any markdown content. Features direct-to-wallet payments, client-side fee splitting, Farcaster mini app integration, and MCP server for AI assistants.
