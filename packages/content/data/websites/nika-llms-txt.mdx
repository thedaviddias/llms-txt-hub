# Nika

Workflow language and runner for AI jobs: one plain-text `.nika.yaml` DAG per job, statically checked before any token is spent (plan, types, secret flows, cost floor), executed by a single Rust binary with hash-chained run traces. Local-first (Ollama, zero keys), MCP on both sides.
