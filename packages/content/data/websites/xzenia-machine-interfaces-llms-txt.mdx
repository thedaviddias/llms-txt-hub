# Xzenia Machine Interfaces

Xzenia exposes machine-native paid intelligence endpoints for autonomous agents and developer tooling.

## What agents can use

- x402-paid company and domain enrichment
- x402-paid market intelligence
- x402-paid contract analysis for leakage-risk review
- OpenAPI and AI plugin manifests for tool ingestion
- llms.txt and llms-full.txt for crawlable machine guidance

## Payment model

The protected API uses x402 payment challenges and settles in USDC on Base mainnet, enabling autonomous machine-to-machine purchases without human checkout.
