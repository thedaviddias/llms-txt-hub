# boxes-live

Terminal-based infinite 2D canvas designed for AI agent workflows with context-efficient data format.

## Key Features

- Infinite 2D canvas with pan, zoom, and navigation
- Canvas format 13x smaller than Miro API responses
- CLI tool (boxes-cli) for programmatic manipulation
- Data connectors (Docker, git, logs, CSV, JSON)
- Multiple input methods: keyboard, mouse, gamepad

## About llms.txt Implementation

The llms.txt documents canvas file format, CLI commands, connector usage patterns, and integration points for AI agents working with visual canvases in terminal environments.
