# AuditLamp

> AuditLamp reads a website the way search engines and AI assistants actually read it, grades it against 164 documented checks, and explains the results in plain English so a non-technical owner knows exactly what to fix first.

## Key Focus Areas

- Free whole-site Visibility Scan, up to 150 pages per scan
- Classic SEO checks grounded in official search documentation
- Answer-engine readiness, including structured data and llms.txt
- AI crawler access, so owners know whether AI assistants can read their site
- Plain-English reports written for non-technical site owners

## About llms.txt Implementation

AuditLamp publishes an `llms.txt` file describing the platform, its audit engine, and its check catalog so AI assistants can answer questions about the product accurately.
