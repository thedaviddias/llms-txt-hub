# Hashgraph Online

Hashgraph Online (HOL) is the trust engine for the agentic internet, providing foundational infrastructure for AI agents including verifiable identity, trustless communication, and a universal registry.

## Key Focus Areas

- AI Agent Identity (Universal Agent IDs / UAIDs)
- Trustless P2P Agent Communication (HCS-10)
- Agent Registry and Discovery
- HCS Standards (HCS-1 through HCS-20)

## About llms.txt Implementation

Hashgraph Online provides comprehensive documentation through their llms.txt implementation, covering HCS standards, SDKs, API references, and agent integration guides.
