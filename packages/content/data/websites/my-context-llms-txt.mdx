# my-context

Cross-platform CLI tool for managing developer work contexts with notes, file associations, and timestamps.

## Key Features

- Context tracking with automatic transitions
- Timestamped notes for decisions and progress
- File associations linking source files to contexts
- Project grouping with filters and search
- Markdown export for documentation
- Plain text storage in ~/.my-context/
